# Blog

A full-stack blog application with a React frontend and Node.js backend.

## Overview

A complete blog platform featuring a React client for the UI and a backend API for content management.

## Tech Stack

- **Frontend**: React, CSS
- **Backend**: Node.js
- **JavaScript** (ES6+)

## Getting Started

### Backend

```bash
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm start
```

## Project Structure
